Going Green Child Theme
http://www.studiopress.com/themes/goinggreen

INSTALL
1. Upload the Going Green child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Going Green theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.
